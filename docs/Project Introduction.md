# JuPITER

## Main Features

* 用户
  * 注册与登录
  * 管理个人信息
  * 接受系统推送
  * 购买门票
  * 查询当前票务信息
  * 查询个人订单信息
  * 接受门票智能推荐
  * 评价交易
  * 竞拍票务
* 管理员
  * 登录
  * 管理票务信息
  * 管理用户信息
  * 查看统计数据
  * 发布竞拍
