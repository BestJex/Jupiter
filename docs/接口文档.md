## 用户

### /login

前端：

```json
{
   "username":"root",
   "password":"root"
}
```

后端：

```json
{
    "status": 0,
    "msg": "登录成功！",
    "data": {
        "userId": 1,
        "userType": 0,
        "username": "root"
    }
}
```

### /register

前端：

```json
{
   "username":"user",
   "password":"user",
   "phone":1
}
```

后端：

```json
{
    "status": -102,
    "msg": "存在相同用户名，请更换用户名",
    "data": null
}
```

或者

```json
{
    "status": 0,
    "msg": "注册成功",
    "data": null
}
```

### /logout

前端：无

后端：

```json
{
    "status": 0,
    "msg": "登出成功！",
    "data": null
}
```

### /checkSession

前端：无

后端：

```json
{
    "status": 0,
    "msg": "登录成功！",
    "data": {
        "userId": 1,
        "username": "root",
        "userType": 0
    }
}
```
